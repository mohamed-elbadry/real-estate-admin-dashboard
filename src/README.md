# Real Estate Admin Dashboard

A responsive Real Estate Admin Dashboard built with Angular and Bootstrap.

## Features

* Dashboard Statistics Cards
* Latest Properties Table
* Properties Listing Page
* Search Properties by Name
* Filter Properties by Status
* Property Details Page
* Property Gallery
* Responsive Layout
* Sidebar Navigation
* Navbar & Footer
* Angular Routing
* Static Data (No Database)

## Technologies

* Angular
* TypeScript
* Bootstrap 5
* Bootstrap Icons

## Installation

```bash
npm install
```

## Run Project

```bash
ng serve
```

Navigate to:

```text
http://localhost:4200
```

## Build Project

```bash
ng build
```
